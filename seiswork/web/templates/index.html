           Leave field empty = not filtered. Max Gap requires station coordinates
          (automatically from config). Min CC (opt-in toggle) cross-correlates every station pair around the pick
          (same QC as event_station_stack_xcorr.ipynb) and drops events whose mean coherence falls below the
          threshold — catches "ghost" events stitched from unrelated picks. Default 0.15 is a conservative outlier
          floor (empirically below the observed range of genuine events on the 7G/Jailolo network — re-tune per network,
          higher = stricter). Requires SDS waveform access and is slower than the other criteria. Results are saved as a
          new session labeled _filter and can be used in all downstream functions.
